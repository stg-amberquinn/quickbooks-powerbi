# QuickBooks Power BI Reporting Integration

## Overview

This application connects QuickBooks Online to Microsoft Power BI to provide reporting, analytics, and business intelligence capabilities.

The integration securely retrieves authorized QuickBooks Online accounting data and makes it available for internal reporting and dashboarding purposes.

## Features

* Secure OAuth 2.0 authentication with QuickBooks Online
* Read-only access to authorized accounting data
* Automated data refresh capabilities
* Microsoft Power BI reporting and analytics
* Internal business intelligence and operational reporting

## Data Usage

The application may access authorized QuickBooks Online data including:

* Customers
* Vendors
* Invoices
* Payments
* Bills
* Accounts
* Expenses
* Journal Entries
* Other accounting records available through the QuickBooks Online API

## Privacy

Data obtained through this application is used solely for authorized reporting and analytics purposes and is not sold or shared with third parties.
